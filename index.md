## 🎯 Project Overview

AI-Resume-Screening is an intelligent web application designed to help both recruiters and job seekers understand how well a resume matches specific job roles. The application uses Natural Language Processing (NLP) to analyze resumes, extract skills, and identify compatibility with job requirements. With this tool, you can visualize strengths, identify skill gaps, and receive personalized learning recommendations.

## ✨ Key Features

- **📄 Resume Upload**: Upload resumes in PDF or DOCX formats. The tool ensures clean text extraction for accurate analysis.
- **🧠 AI Skill Extraction**: The application identifies skills using NLP technology, specifically through spaCy. It uses a custom skill database for better accuracy.
- **🎯 Job Role Matching**: Compare resumes against five predefined job roles. The system calculates a match percentage for each role, giving you clear insights.

Once you have installed AI-Resume-Screening, follow these steps to analyze a resume:

1. **Upload Your Resume**: Click the "Upload" button within the application. Choose a resume file in PDF or DOCX format.
